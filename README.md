# Разработка

Тут будет описан процесс разработки

## Цели

Сцена леса:
- Деревья
- Ограничение густым лесом по бокам (просто стена)

## Структура папок

Папки устроены следующим образом:
- Assets
  - Gift2 - основная папка игры
    - Scenes - содержит сцены (сюда их помещать)
    - Prefabs - содержит заготовки объектов (префабы)
    - Resources - содержит спрайты, звуки, анимации
    - Configs - содержит файлы конфигурации
    - Scripts - содержит скрипты
  - Другие папки (сторонние плагины или скрипты из других проектов)

## Общие правила

Все повторяющиеся объекты (деревья, камни и т.д.) должны быть вынесены в отдельный префаб.

Пол и стены, ограничивающие локацию, делается через TileMap.
